# Feature Voting UI

## Configuration

To configure the API URL, edit `config/constants.js` and update the `API_URL` value.

For iOS simulator: `http://localhost:3000/api`
For physical device: Use your computer's IP address (e.g., `http://192.168.1.100:3000/api`)

## Running

```bash
npm run ios
```

Make sure the backend API is running on port 3000.
